# Screenshot delle guide utente

Questa cartella raccoglie gli **screenshot** referenziati dalle guide in [`docs/user/`](../../user/).

## Come contribuire

- Cattura gli screenshot **su Windows** (l'app è Windows-first: aspetto, font e finestre reali).
- Usa nomi descrittivi e stabili, es. `selettore-lingua.png`, `wizard-passo-1.png`,
  `assistente-tab.png`, `assistente-banner-conferma.png`.
- Referenziali dalle guide sostituendo i segnaposto `[screenshot: …]` con l'immagine, es.
  `![Selettore lingua](../assets/screenshots/selettore-lingua.png)`.
- **Non** includere dati sensibili negli screenshot: **oscura** bot token, API key, chat ID reali e
  qualsiasi percorso/credenziale prima di salvare l'immagine.

> Regola del repo: quando una PR cambia schermate/pulsanti/flussi della UI, gli screenshot qui e le
> guide in `docs/user/` vanno aggiornati nello stesso PR.
