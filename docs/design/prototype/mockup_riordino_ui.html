XTrader Bridge — Concept di riordino UI

Idea di organizzazione (non è l'app vera): stesse funzioni, meno confusione, più ordine. L'aspetto può cambiare — la logica resta la tua.

1 · L'idea in una riga

Oggi le schede sono organizzate per oggetto (Parser, Provider, Mapping, Chat…): per configurare un canale salti tra 4 schede. Il concept le riorganizza per "cosa vuoi fare", in 4 gruppi, e mette le mappature dentro il Parser (dove si accendono).

5 · Cosa cambia in pratica (riassunto)

  • Rinominata la colonna «Provider» del dizionario nomi → «Come lo scrive il canale» (fine ambiguità).
  • Mappature dentro il Parser (riquadro «Traduzioni attive»), non più in una scheda a parte.
  • 4 gruppi per flusso invece di 7 schede piatte.
  • Colonna «Traduzioni» in Chat sorgenti + schermata Riepilogo per lo stato a colpo d'occhio.
  • Parser meno denso: avanzate (Trasformazione/Value-map) nascoste dietro un tasto.
  • MultiMarket / MultiSelection (output multi-riga) preservati, dentro «② Lettura messaggi».
  • Le funzioni e le regole di sicurezza restano identiche: cambia solo l'organizzazione.
